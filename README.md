# Система мониторинга товаров (Лабораторная работа №5)

##  Описание проекта
Программа для работы с базой данных SQLite, реализующая систему мониторинга товаров. Приложение позволяет подключаться к локальной базе данных, просматривать, добавлять и редактировать информацию о товарах.

##  Функциональность
- **Подключение к БД SQLite** (*.db файлы)
- **Просмотр таблиц** с товарами различных категорий
- **CRUD операции**:
  -  Добавление новых товаров
  -  Редактирование характеристик товаров
  -  Просмотр всех записей
- **Фильтрация данных**:
  - По производителю
  - По количеству товара (≤ 2)
- **Статистические расчеты** для числовых полей:
  - Минимальное значение
  - Максимальное значение
  - Среднее арифметическое
  - Сумма значений

##  Технологии
- **Язык программирования**: C#
- **Платформа**: .NET Framework / Windows Forms
- **Технология доступа к данным**: ADO.NET
- **СУБД**: SQLite
- **NuGet пакеты**: System.Data.SQLite

##  Структура базы данных
База данных содержит минимум две таблицы с товарами (например, "Компьютерные комплектующие" и "Бытовая техника") со следующими полями:
- Идентификатор
- Наименование товара
- Производитель
- Цена
- Количество
- Дополнительные поля (по желанию)

В каждой таблице не менее 10 записей.

##  Запуск приложения
1. Открыть решение в Visual Studio
2. Восстановить NuGet пакеты (System.Data.SQLite)
3. Скомпилировать и запустить проект
4. Нажать "Подключить БД" и выбрать файл базы данных (*.db)
5. Выбрать таблицу для работы

##  Пример использования
1. Подключение к БД → выбор таблицы → просмотр данных
2. Выбор поля → расчет статистики (мин/макс/среднее/сумма)
3. Фильтрация по производителю или остаткам товара
4. Добавление/редактирование товаров через интерфейс
